CHECK : How to use samples

1. Please check the option listed at the bottom of the top page first on web server in robot.
Some product line-ups (for example Thouzer Basic) can NOT use these applications that marked P)MQTT/OFF in the options.
In some simpler product line-ups, some interfaces, such as the MQTT interface and parameter licensing, are not supported.

2. Please enable Cookie on your browser.
If enable Cookie, for example "address" is displayed fine on the Application Folder.

3. Check the string at the top of the sample application.
This indicates the connection status.
If MQTT on Websocket is connected successfully, the unit serial number will be displayed.

4. For other trouble shooting.
Check with the distributor or the contact listed in the instruction manual.
